# Contribuyendo a BRE-B PSE Recaudos App

## Cómo Contribuir

1. **Fork** el repositorio
2. **Crea una rama** para tu feature (`git checkout -b feature/AmazingFeature`)
3. **Commit** tus cambios (`git commit -m 'Add some AmazingFeature'`)
4. **Push** a la rama (`git push origin feature/AmazingFeature`)
5. **Abre un Pull Request**

## Estándares de Código

### Backend (Go)
- Usa `gofmt` para formatear código
- Escribe tests para nuevas features
- Comenta funciones públicas
- Usa nombres descriptivos para variables

### Frontend (Vue 3)
- Usa `npm run lint` para verificar el código
- Componentes en PascalCase
- Props validadas con tipos
- Mantén archivos menores a 300 líneas

## Reportar Bugs

Abre un issue con:
- Descripción clara del bug
- Pasos para reproducirlo
- Comportamiento esperado vs actual
- Screenshots si aplica

## Sugerencias de Features

Crea un issue con:
- Título descriptivo
- Descripción detallada
- Casos de uso
- Beneficios esperados

¡Gracias por contribuir!
